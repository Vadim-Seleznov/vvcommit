# Installation

## Requirements

- Python 3.8 or higher
- Git installed and available in PATH

## Clone Repository

```bash
git clone https://github.com/Vadim-Seleznov/vvcommit.git
cd vvcommit
```

and best practice would be adding an alias in your .bashrc or .zshrc:
```bash
alias vvcommit="python /absolute/path/to/vvcommit.py"
```

## VERIFY Installation
```bash
vvcommit help
```
